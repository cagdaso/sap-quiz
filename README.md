# SAP Live Quiz

A Mentimeter-style live audience quiz powered by Firebase + GitHub Pages.
**No server needed** - works on any device worldwide.

## Live URLs
- Host: https://cagdaso.github.io/sap-quiz/host.html
- Vote: https://cagdaso.github.io/sap-quiz/vote.html

## How to use
1. Open host.html in your browser
2. Show the QR code to your audience
3. Click **Start Round** - participants tap A/B/C/D
4. Click **End Round** - winner is revealed
5. Click **Next Round** - repeat

## Firebase
- Project: simplepollbyxpr
- Database: https://simplepollbyxpr-default-rtdb.europe-west1.firebasedatabase.app
